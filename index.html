Notable Projects

Below you can find some of my personal projects.

ClangTuner: A C++ attribute extension for optimization and offloading.

Clang is extended with a set of CXX-11 attributes that apply to parallel for loops. for loops having this attribute will be compiled to standard MLIR code, and will be passed to the MLIR optimizer. The optimized for loop will then be lowered to LLVM IR and linked with the rest of the program.

Quack Compiler

Quack is an object-oriented language with single inheritance and type inference. Bison and Flex are used for generating the Abstract Syntax tree. The produced AST is then visited to generate LLVM-IR code with the LLVM builder. The compiler is also capable of generating C code.

Duck Chat

A command line based chat-room application using Linux's networking stack. This application uses the client-server architecture. Multiple users can sign-in, create/delete/join channels and chat with other users in those channels. The system can support multiple servers and simple load balancing is implemented through flooding. The clients communicate with the servers using the duck-chat protocol, and servers talk to each other with the duck-chat-s2s protocol. Communications happen over UDP.
